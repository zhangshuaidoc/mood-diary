# 心情日志 - 微信小程序

一款极简的心情记录小程序，帮助你轻松记录每一天的情绪变化。

## 功能特性

### MVP功能 (V1.0)

- ✅ **心情选择** - 8种emoji心情选择器
- ✅ **文字记录** - 一句话描述（可选，限100字）
- ✅ **日历视图** - 月历展示，每天显示心情emoji
- ✅ **本地存储** - 微信小程序本地缓存
- ✅ **记录编辑** - 修改/删除记录

## 技术栈

- **前端框架**: 微信小程序原生开发
- **UI组件**: 自定义组件
- **数据存储**: wx.setStorageSync / wx.getStorageSync
- **设计风格**: 清新简洁，绿色主题

## 快速开始

### 1. 下载项目

```bash
# 克隆或下载项目到本地
```

### 2. 导入微信开发者工具

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目目录
4. 填写AppID（可使用测试号）
5. 点击"导入"

### 3. 运行项目

- 在微信开发者工具中点击"编译"即可运行
- 可以在模拟器中预览效果
- 也可以在真机上预览

## 数据结构

### 心情记录

```javascript
{
  id: String,           // 记录唯一ID（时间戳）
  date: String,         // 日期 "2026-03-21"
  mood: String,         // 心情类型 "happy"
  moodEmoji: String,    // 心情emoji "😊"
  text: String,         // 文字描述（可选）
  createTime: Number,   // 创建时间戳
  updateTime: Number    // 更新时间戳
}
```

### 本地存储

```javascript
// Key: "mood_records"

wx.setStorageSync('mood_records', records)
```

## 注意事项

1. **数据存储**: 所有数据存储在微信本地缓存，卸载小程序会丢失数据
2. **隐私保护**: 数据仅存储在本地，不会上传到服务器
3. **图标替换**: 当前使用占位图标，建议替换为设计好的PNG图标
